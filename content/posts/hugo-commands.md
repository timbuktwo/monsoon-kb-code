## Most commonly used Hugo commands:

#### New post:  
```
hugo new posts/post-title.md
```

#### Launch local web server:  
```
hugo serve
```
#### Generate static website files & include content marked as draft:
```
hugo -D
``` 
#### New page:
```
hugo new page-title.md
```
